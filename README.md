# ypuf

**Your Pages, Unburdened & Findable.**

A Chrome extension that lets you stop *managing* tabs. ypuf quietly clears the
tabs you've stopped caring about and gives you instant, full-text recall of
everything you've ever had open — so you can close everything and lose nothing.

Tabs are open loops. ypuf treats them like one: let go of what's dead (the
*ebb*), recall what matters the moment you need it (the *flow*).

> **Status: pre-implementation.** This repo currently holds the product and
> technical decisions from the ideation session. Start at
> [`docs/CONTEXT.md`](docs/CONTEXT.md) — it has everything needed to spec or
> build from.
